# hubot-shortcut

A macro system for hubot.

See [`src/shortcut.coffee`](src/shortcut.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-shortcut --save`

Then add **hubot-shortcut** to your `external-scripts.json`:

```json
[
  "hubot-shortcut"
]
```
